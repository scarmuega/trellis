# Automated bookkeeping

The chosen means: ledgerloom ingests transactions and categorizes them
automatically, so the studio owner's books stay current without touching them.
It induces two core subdomains — categorizing transactions accurately
(`problem/categorization-accuracy.md`) and the accountant partnerships that
vouch for the result (`problem/accountant-partnerships.md`) — plus a supporting
web presence (`problem/site-presence.md`) and generic invoice emailing
(`problem/invoice-emailing.md`).
